🚀 HOW TO RUN COGNIPLAN (PRODUCTIVITY TRACKER)
Location: D:\Projects\Tracker
OPTION 1: THE EASY WAY (AUTOMATIC)
Double-click the START_APP.bat file in the root folder.
This script is configured to:

Activate the Python Virtual Environment (.venv).

Start the FastAPI backend server.

Launch your browser automatically to http://127.0.0.1:8000.

OPTION 2: THE MANUAL WAY (VS CODE / POWERSHELL)
If you prefer running it manually to see the logs and debug, follow these steps:

Open Terminal in the Tracker folder.

Activate the Environment:

PowerShell
.\.venv\Scripts\Activate.ps1
Enter the Backend Directory:

PowerShell
cd backend
Start the Server:

PowerShell
uvicorn main:app --reload
View the App: Open your browser and go to http://127.0.0.1:8000.

📂 IMPORTANT: FOLDER STRUCTURE
To ensure the app runs correctly, keep the files organized as follows:

/backend: Contains main.py, models.py, and your .env (connected to Supabase).

/docs: Contains your frontend (index.html, style.css, app.js).

Note: We use the docs folder instead of frontend so the app works both locally and on GitHub Pages.

☁️ DATABASE STATUS
Provider: Supabase (PostgreSQL)

Connection: Managed via the DATABASE_URL in your backend/.env.

Auto-Sync: Tables are automatically checked/created every time the backend starts.

🛠️ TROUBLESHOOTING
"ModuleNotFoundError": If you see a missing module (like psycopg2), run pip install -r backend/requirements.txt.

"Port 8000 in use": If the server won't start, close any other Python windows or run uvicorn main:app --reload --port 8001.
